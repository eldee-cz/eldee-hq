Vybraný objekt

Vyber nahoře nástroj. 🪣 Kyblík = klikni do plochy a vyleje barvu (ohraničí ji obrysy). 🖌 Štětec = maluj tažením. ✂ Klonovat = orámuj kus obrázku → kopie. Razítka přidají hotový kus.
📂 Načíst maskota
Box — dvě čisté štulpny:
1. 🪣 Kyblík + barva tma boxu → klikej do špinavých štulpen, až je box prázdný
2. + Levá štulpna → polož, uprav velikost/otočení
3. ⧉ Duplikovat → druhou vedle (nebo ↔ Zrcadlit pro pár)

Límeček fotbalisty:
1. Fotbalista nahoře vpravo
2. 🪣 Kyblík + barva bílá → klikni do límečku
